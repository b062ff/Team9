ファイルについての説明
sample.php
DBを使って投稿保存が行えるプログラムです。
Googleのcode-sandboxではDBがおそらく使えないので、こちらは使用しないでください。

sample_noDB.php
DBを使ってないので投稿と保存は行えません。
音声再生などは行えるのでgoogleのsandboxでもおそらく使用できます。cssやjsの修正をsandboxで行いたい場合は、こちらのソースコードを改編してください。
クラスの追加や表示形式の変更などは自由に行ってください。

style.css
特に記述をしてないので、自由に書いてください。

index.js
現状書いてあるスクリプトはバグがあるので、消しても大丈夫です。

test.php
設計図の様なページを切り替えて投稿したり、再生したり、セリフをそれぞれの設定で決定するようにしました。
音声再生機能で修正が必要様です。
